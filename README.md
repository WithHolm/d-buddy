# d-buddy
tui for dbus monitor using rust

## app arguments

* log: `--log` . also remember to set log level in env: `RUST_LOG=d_buddy=trace`
* enable debug ui: `--debug-ui`
* start and make sure it runs: `--check`



## Build
```
make build
```

## Run
```
make run
```

## Clean
```
make clean
```
